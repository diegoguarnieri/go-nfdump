# go-nfdump

go-nfdump read and decode nfdump files. This preliminar version was tested with NetFlow v5.

## Installation
go get github.com/diegoguarnieri/go-nfdump

## Future
* Test with other NetFlow versions
* ipv6 support
* Encode module
